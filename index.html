Disguise Detection

The Verified Vision DDS01 (Disguise Detection System) can be used to verify that people are not trying to hide their true identity.

Read more

The Problem

Hollywood quality latex masks are now available online. They are affordable and hard to distinguish from real faces. Every week criminals worldwide are using masks for robberies, and there are reported cases of people using high-end masks to pass through borders.

Read more

The Solution

The DDS01 is an inexpensive box that uses recent advances in multispectral imaging to determine if a person is disguised. The DDS01 can be standalone or networked. It will work on a counter, or in areas of CCTV surveillance such as entries or corridors.

Read more

Applications

The DDS01 can be used wherever face identity checks are required, such as immigration, policing, and driver's license authorities. It can also enhance the security of existing surveillance systems at banks, jewellery stores, casinos, military sites, power stations, etc..

Read more

Anti-Spoofing for Face Recognition Systems

It is widely known in the security community that all biometric systems have vulnerabilities that allow people to be falsely verified. This is a particular problem for face recognition systems, which can often be spoofed using relatively simple techniques.
